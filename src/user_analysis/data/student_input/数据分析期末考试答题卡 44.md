### 第2题 解释准确率和召回率的区别，并写出它们的计算公式。（10分） 得分：5
**学生答案**：  
准确率是正确预测的比例，召回率是找出正例的比例。  
准确率 = TP/总数  
召回率 = TP/正例数

**正确答案**：  
准确率是所有预测中正确的比例，召回率是实际正例中被正确预测的比例。  
准确率 = (TP+TN)/(TP+TN+FP+FN)  
召回率 = TP/(TP+FN)

**答案解析**：  
概念理解基本正确，但公式表述不准确。

## 六、编程题（共1题，共10分） 得分：4

### 题目：使用Python编写一个函数，计算数据集的均值、中位数和众数。（10分） 得分：4

**学生答案**：
```python
def calc_stats(data):
    mean = sum(data) / len(data)
    # 中位数和众数不会计算
    median = data[len(data)//2]
    return mean, median, 0
```

**正确答案**：
```python
import numpy as np
from scipy import stats

def calc_stats(data):
    mean = np.mean(data)
    median = np.median(data)
    mode = stats.mode(data)[0][0]
    return mean, median, mode
```

**答案解析**：  
只能正确计算均值，中位数计算错误，众数未实现，编程基础薄弱。
